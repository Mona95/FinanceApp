## Finance App

#### How to run project

- Unzip the folder.
- run `npm install`.
- run `npm start`.

<hr/>

#### Used Dependencies

- @material-ui/core
- @material-ui/icons
- axios
- redux
- react-redux
- redux-thunk
<hr/>
